# ETF Atlas v1 기능 명세

## 프로젝트 목표

ETF 정보 관리를 통한 인사이트 제공
- 장중 실시간 대응은 v1 범위 외
- 매일 배치 처리로 데이터 수집/분석

---

## ETF 유니버스

### 포함

| 조건 | 설명 |
|------|------|
| 국내 주식형 ETF | 국내 상장 주식으로 구성 |
| 해외 주식형 ETF | S&P500, 나스닥, 해외 섹터 등 |
| 순자산 1,000억 이상 | 유동성 확보 |

### 제외

| 조건 | 이유 |
|------|------|
| 합성/파생 ETF | 실물 구성종목 없음 |
| 레버리지/인버스 ETF | 파생 상품 |
| 커버드콜 ETF | 옵션 전략 포함 |
| 채권/원자재/금/통화 ETF | 주식 종목 아님 |

> 상세 필터링 로직: [data-mapping.md](./data-mapping.md#etf-유니버스-정의)

## 핵심 인사이트 3가지

### 1. 종목 역추적 (Stock → ETF)
> "삼성전자에 투자하고 싶은데, 어떤 ETF를 사면 될까?"

| 제공 정보 | 예시 |
|-----------|------|
| 특정 종목 포함 ETF 리스트 | 삼성전자 담은 ETF 15개 |
| ETF별 해당 종목 비중 | KODEX 200: 25%, TIGER 반도체: 18% |
| 비중 순위 정렬 | 가장 많이 담은 ETF → 적게 담은 ETF |

### 2. 포트폴리오 변화 감지 (워치리스트 기반)
> "운용사들이 실제로 사고 파는 종목은?"

- **패시브 ETF**: 인덱스 추종이므로 변화 추적 불필요
- **액티브 ETF**: 사용자가 워치리스트에 등록하여 모니터링

#### 운용사 판단 감지 기준

단순 비중 변화는 주가 변동으로 발생할 수 있어 신뢰도가 낮음.
**운용사의 의도적 판단**만 추적:

| 시그널 | 설명 | 신뢰도 |
|--------|------|--------|
| **신규 편입** | 없던 종목을 새로 추가 | 높음 |
| **완전 제외** | 있던 종목을 포트폴리오에서 퇴출 | 높음 |
| **비중 5%p 이상 변화** | 주가 변동만으로 설명 어려움, 의도적 매수/매도 | 높음 |

```
예시:
• 삼성전자 10% → 16% (+6%p) → 운용사 적극 매수 신호
• SK하이닉스 8% → 2% (-6%p) → 운용사 적극 매도 신호
• NAVER 5% → 6% (+1%p) → 무시 (주가 변동 가능성)
```

#### v2 고려사항
- **상대 비중 변화**: ETF 순자산 변화율 대비 종목 보유액 변화율 비교
- 더 정밀한 운용사 판단 감지 가능

### 3. AI 종목 추천
> "워치리스트 ETF들이 공통적으로 담는 종목은?"

- 워치리스트 내 액티브 ETF 포트폴리오 변화 분석
- 신규 편입/제외/비중 대폭 변화 기반 매수/매도 신호 도출

---

## 워치리스트 기능

```
[워치리스트]
├── 사용자가 모니터링할 ETF 직접 등록
├── 주로 액티브 ETF 대상
└── 등록된 ETF만 포트폴리오 변화 추적
```

| 기능 | 설명 |
|------|------|
| ETF 추가/삭제 | 모니터링 대상 관리 |
| 기간별 수익률 표시 | 전일, 1주일, 1달 |
| 변화 알림 | 신규편입/제외/비중 5%p 이상 변화 시 알림 |

---

## v1 성공 기준

```
✅ 사용자가 종목명 입력 → 관련 ETF 리스트 즉시 조회
✅ 특정 ETF의 포트폴리오 변화 확인 (신규편입/제외/비중 5%p 이상)
✅ 워치리스트에 액티브 ETF 등록/삭제 + 수익률 한눈에 확인
✅ 워치리스트 기반 AI 종목 추천 (운용사 판단 시그널 기반)
```
