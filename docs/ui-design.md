# ETF Atlas UI 설계

## 화면 구성

| # | 화면 | 핵심 기능 |
|---|------|----------|
| 1 | 종목 검색 | 종목명 → ETF 리스트 (비중순) |
| 2 | ETF 상세 | 캔들차트 + 구성종목 + 포트폴리오 변화 |
| 3 | 워치리스트 | ETF 추가/삭제 + 수익률 + 변화 알림 |
| 4 | AI 종목추천 | 신규편입/제외/비중 5%p 변화 기반 신호 |

---

## 전체 레이아웃

```
┌─────────────────────────────────────────────────────────────┐
│  ETF Atlas                                       [설정]     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│   [종목 검색]  [ETF 검색]  [워치리스트]  [AI 추천]          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

---

## 1. 종목 검색 (역추적)

### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  종목으로 ETF 찾기                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [삼성전자_________________________] [검색]                 │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  삼성전자 (005930) 를 담은 ETF: 32개                        │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ETF명                      │ 비중   │ 유형   │ 총자산      │
│  ────────────────────────────────────────────────────────── │
│  KODEX 반도체               │ 28.5% │ 패시브 │ 1.2조       │
│  TIGER 200                  │ 25.1% │ 패시브 │ 3.5조       │
│  TIMEFOLIO 코리아플러스액티브│ 18.2% │ 액티브 │ 800억    *  │
│  ...                                                        │
│                                                             │
│  [비중순 v]  [액티브만]  [자산규모 1000억+]                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
  * = 워치리스트 등록됨
```

### 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| StockSearchInput | 종목명 검색 + 자동완성 |
| ETFResultTable | ETF 리스트 테이블 (정렬, 필터) |
| FilterChips | 액티브만, 자산규모 필터 |

### 인터랙션

- 종목명 입력 시 자동완성 드롭다운
- ETF 행 클릭 → ETF 상세 페이지 이동
- 별표(*) 클릭 → 워치리스트 추가/삭제

## 2. ETF 상세

### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  <- TIMEFOLIO 코리아플러스배당액티브          [* 워치리스트]│
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  유형: 액티브    운용사: 타임폴리오    총자산: 856억        │
│  설정일: 2021.03.15                                         │
│                                                             │
│  수익률:  전일 +0.8%  │  1주 +2.3%  │  1달 +5.1%           │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  가격 차트                                  [1주|1달|3달|1년]│
├─────────────────────────────────────────────────────────────┤
│  ┌────────────────────────────────────────────────────┐    │
│  │     │                          ┌─┐                 │    │
│  │     │                    ┌─┐   │ │  ┌─┐            │    │
│  │     │              ┌─┐   │ │   │ │  │ │  ┌─┐       │    │
│  │     │        ┌─┐   │ │   │ │   │ │  │ │  │ │       │    │
│  │     │  ┌─┐   │ │   │ │   │ │   │ │  │ │  │ │       │    │
│  │  12,000 ──────────────────────────────────────     │    │
│  │     1/1     1/8    1/15   1/22   1/29              │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  구성 종목                │  포트폴리오 변화    [1달v|3달|6달]│
├──────────────────────────┼──────────────────────────────────┤
│                          │                                  │
│  종목명      │ 비중      │   [신규] 한화에어로스페이스       │
│  ─────────────────────   │   [제외] 엔씨소프트               │
│  삼성전자    │ 15.2%     │   ^ SK하이닉스  8%->14% (+6%p)   │
│  SK하이닉스  │ 14.0%     │   v 카카오     10%->4% (-6%p)    │
│  현대차      │  8.5%     │                                  │
│  NAVER      │  6.2%     │                                  │
│  ...                     │                                  │
│                          │                                  │
└─────────────────────────────────────────────────────────────┘
```

### 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| ETFHeader | ETF 기본정보, 수익률, 워치리스트 버튼 |
| PriceChart | 캔들차트 (Recharts) |
| HoldingsTable | 구성종목 테이블 |
| ChangesPanel | 포트폴리오 변화 리스트 |
| PeriodSelector | 기간 선택 탭 |

### 인터랙션

- 워치리스트 버튼 → 등록/해제 토글
- 기간 탭 클릭 → 차트/변화 데이터 갱신
- 종목명 클릭 → 해당 종목 역추적 페이지 이동

## 3. 워치리스트

### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  * 워치리스트                               [+ ETF 추가]    │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  등록된 ETF: 5개                                            │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ETF명                     │ 전일   │ 1주   │ 1달   │       │
│  ────────────────────────────────────────────────────────── │
│  TIMEFOLIO 코리아플러스     │ +0.8% │ +2.3% │ +5.1% │  [x]  │
│  KODEX K-미래차액티브       │ +1.2% │ +3.1% │ +7.2% │  [x]  │
│  TIGER AI코리아액티브       │ -0.3% │ +1.5% │ +4.8% │  [x]  │
│  ARIRANG 고배당주액티브     │ +0.5% │ +0.8% │ +2.1% │  [x]  │
│  HANARO 글로벌럭셔리S&P     │ +0.2% │ -0.5% │ +1.3% │  [x]  │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  최근 포트폴리오 변화                                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  * TIMEFOLIO 코리아플러스: [신규] 한화에어로스페이스 (1/15) │
│  * KODEX K-미래차: SK하이닉스 8%->14% (+6%p) (1/14)         │
│  * TIGER AI코리아: [제외] 엔씨소프트 (1/13)                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### ETF 추가 모달

```
┌─────────────────────────────────────────────┐
│  ETF 검색                              [x]  │
├─────────────────────────────────────────────┤
│                                             │
│  [________________] [검색]                  │
│                                             │
│  검색 결과:                                 │
│  ┌─────────────────────────────────────┐   │
│  │ KODEX 2차전지액티브           [추가] │   │
│  │ TIGER 글로벌리튬액티브         [추가] │   │
│  │ ...                                  │   │
│  └─────────────────────────────────────┘   │
│                                             │
└─────────────────────────────────────────────┘
```

### 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| WatchlistTable | 워치리스트 테이블 + 수익률 |
| AddETFModal | ETF 검색/추가 모달 |
| ChangeAlerts | 포트폴리오 변화 알림 리스트 |
| DeleteButton | ETF 삭제 버튼 |

### 인터랙션

- [+ ETF 추가] → 모달 오픈
- [x] 버튼 → 워치리스트에서 삭제 (확인 팝업)
- ETF명 클릭 → ETF 상세 페이지 이동
- 변화 알림 클릭 → 해당 ETF 상세 페이지 이동

## 4. AI 종목 추천

### 와이어프레임

```
┌─────────────────────────────────────────────────────────────┐
│  AI 종목 추천                                   [새로고침]  │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  워치리스트 ETF 포트폴리오 변화 기반 분석                   │
│  분석 기간: 최근 1개월                                      │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  ^ 매수 신호 종목                                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  종목명           │ 시그널              │ ETF 수 │ 신호강도 │
│  ────────────────────────────────────────────────────────── │
│  한화에어로스페이스 │ 신규편입 2건        │ 2/5개  │ ***      │
│  SK하이닉스       │ 비중 +6%p, +5%p    │ 2/5개  │ ***      │
│  현대차           │ 신규편입 1건        │ 1/5개  │ **       │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  v 매도 신호 종목                                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  종목명           │ 시그널              │ ETF 수 │ 신호강도 │
│  ────────────────────────────────────────────────────────── │
│  엔씨소프트       │ 완전제외 2건        │ 2/5개  │ ***      │
│  카카오           │ 비중 -6%p          │ 1/5개  │ **       │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  AI 인사이트                                                │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  "워치리스트 내 액티브 ETF 운용사들이 방산/반도체 섹터에    │
│   적극적으로 진입하고 있습니다. 한화에어로스페이스는        │
│   2개 ETF에서 신규 편입되었고, SK하이닉스는 2개 ETF에서     │
│   비중을 5%p 이상 늘렸습니다.                               │
│   반면 게임 섹터(엔씨소프트)는 퇴출 움직임이 보입니다."     │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| SignalTable | 매수/매도 신호 테이블 |
| SignalStrength | 신호 강도 표시 (별점) |
| InsightPanel | AI 인사이트 텍스트 |
| RefreshButton | 분석 새로고침 |

### 인터랙션

- [새로고침] → AI 분석 재실행 (로딩 표시)
- 종목명 클릭 → 해당 종목 역추적 페이지 이동

```
│  종목 검색   │────>│  ETF 상세    │────>│  워치리스트  │

                     └──────────────┘     └──────────────┘
                                                 │
                                                 v
                                          ┌──────────────┐
                                          │  AI 종목추천 │
                                          │  (포트변화   │
                                          │   기반)      │
                                          └──────────────┘
```

## 공통 컴포넌트

| 컴포넌트 | 설명 |
|----------|------|
| Header | 상단 네비게이션 |
| SearchInput | 자동완성 검색창 |
| DataTable | 정렬/필터 테이블 |
| PeriodTabs | 기간 선택 탭 |
| Badge | 유형 뱃지 (액티브/패시브) |
| Modal | 모달 다이얼로그 |
| Toast | 알림 토스트 |

## 반응형 고려사항

| 화면 | Desktop | Mobile |
|------|---------|--------|
| 종목 검색 | 테이블 전체 표시 | 카드 레이아웃 |
| ETF 상세 | 2컬럼 (구성종목 / 변화) | 1컬럼 탭 전환 |
| 워치리스트 | 테이블 전체 표시 | 수익률 축약 표시 |
| AI 추천 | 테이블 전체 표시 | 카드 레이아웃 |
